## Возможности

- Добавление двигателей разных типов
- Удаление по индексу
- Вывод всех элементов
- Сравнение двух объектов
- Валидация пользовательского ввода

## Используемые технологии

- Java
- Коллекции (ArrayList)
- Наследование
- Полиморфизм
- Factory pattern

## Структура классов

- Engine — базовый класс
- InternalCombustionEngine — ДВС
- DieselEngine — дизельный двигатель
- JetEngine — реактивный двигатель
- EngineFactory — фабрика создания объектов
